# 我的房间

这是一个有趣的网站，使用Next.js构建。第一次访问的用户会收到惊喜，还有一个小游戏可以自娱自乐。

## 功能

- **惊喜动画**：第一次访问时弹出“我知道你”消息
- **小游戏**：点击按钮得分游戏
- **现代UI**：使用Tailwind CSS的游戏风格设计

## 运行项目

首先，安装依赖：

```bash
npm install
```

然后，启动开发服务器：

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看结果。

## 部署

要部署到你的域名 myroom.ccwu.cc，你可以使用Vercel或其他平台。

1. 推送到Git仓库
2. 在Vercel中连接仓库
3. 设置域名

更多信息请查看 [Next.js部署文档](https://nextjs.org/docs/app/building-your-application/deploying)。
